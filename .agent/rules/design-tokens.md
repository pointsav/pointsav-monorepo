# Design Token Workflow

All design work and UI styling developed for `app-mediakit-knowledge` must be extracted and formally exported to the appropriate design system and media asset repositories.

## Routing Rules
1. **Generic Design Tokens:** All structural UI elements, spacing variables, layout grids, and generic typography rules must be extracted and committed to `/pointsav-design-system`.
2. **Branded Tokens (PointSav):** Any PointSav-specific assets, logos, brand colors, or specialized visual themes must be committed to `/pointsav-media-assets`.
3. **Branded Tokens (Woodfine):** Any Woodfine-specific assets, logos, brand colors, or specialized visual themes (used for `projects.woodfinegroup.com` and `corporate.woodfinegroup.com`) must be committed to `/woodfine-media-assets`.
