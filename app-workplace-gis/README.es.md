# Workplace GIS

Visor GIS de escritorio soberano para datos de inteligencia de ubicación de PointSav.
Se conecta a gis.woodfinegroup.com o a un servidor de teselas local a través de WireGuard PPN.

**Plataforma:** macOS 10.13 High Sierra y versiones posteriores  
**Tecnología:** Tauri v1.7 (WebView cargando el visor de teselas MapLibre GL)  
**Licencia:** Apache-2.0
