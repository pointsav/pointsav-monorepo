# Workplace GIS

Sovereign desktop GIS viewer for PointSav location intelligence data.
Connects to gis.woodfinegroup.com or a locally-running tile server over WireGuard PPN.

**Platform:** macOS 10.13 High Sierra and later  
**Stack:** Tauri v1.7 (WebView loading MapLibre GL tile viewer)  
**Licence:** Apache-2.0

## Wave 2 — foundation scaffold

This application is in Wave 2 (foundation committed 2026-05-27).
Active development begins after Wave 1 trio ships.

## Planned capabilities

- View GIS cluster map (T1/T2/T3 layers from gis.woodfinegroup.com)
- Connect to local tile server over WireGuard PPN
- Configurable endpoint
- Export current view as PNG/PDF
