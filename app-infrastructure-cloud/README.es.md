# app-infrastructure-cloud

Cartucho de infraestructura para nodos en la nube (GCP foundry-workspace).

Ejecuta en `os-infrastructure`. Desplegado como `fleet-infrastructure-cloud` en el nivel
de presentación y `fleet-infrastructure-cloud-1` en la Red Privada de PointSav.

Este nodo actúa como centro de comando: aloja los archivos Totebox, el Doorman SLM y
todo el estado de la Sesión de Comando del espacio de trabajo Foundry.

Parte de la familia `app-infrastructure-*`.
