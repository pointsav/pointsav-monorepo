# 🏗️ SYSTEM-SLM (The Semantic Router)
**Entity:** PointSav Digital Systems (The Vendor)
**Taxonomy:** Tier-1 Core Component
**Status:** Active Engineering

## I. ARCHITECTURAL MANDATE
This component serves as the Semantic Command Router. Utilizing highly quantized, sub-billion parameter Small Language Models (SLMs), it translates natural English intent into strict binary UDP commands. 

## II. THE FIDUCIARY GATEWAY (HUMAN-IN-THE-LOOP)
To prevent autonomous execution liabilities, `system-slm` is mathematically prohibited from directly broadcasting to the network mesh. Its absolute function is to output a proposed JSON payload. This payload must be routed back to the terminal interface for explicit human verification and authorization before execution.
