Design System — Interface OS

Interface

The Interface OS aggregates from the Toteboxes over the private network and exposes a proprietary interface stack sold as separate chargeable products — each one a line item, each one billed where the data leaves the customer’s own archives.

Why this is the revenue layer
The Totebox OS holds the customer’s data and is theirs. The interfaces are what we charge for and keep proprietary — the Interface OS aggregates across multiple Toteboxes over the private network, and each product above (Command Centre API, Data Engine, Data Marketplace, BIM Server, Building ID Server) is licensed on its own. The planned seL4-isolation work and the intended Sovereign Data Foundation sit beneath this layer; neither ships today.
