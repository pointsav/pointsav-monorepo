Bookkeeping & Minute Books

A conventional bookkeeper keeps numbers in a database and points each row at a document by a reference number. Here the document is the record: every source file is written once and never altered (write-once), and its name carries the full chart-of-accounts reference — so the file system itself is the index and there is no second database lookup to keep in sync.
