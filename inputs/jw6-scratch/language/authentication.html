Your notes
P1 Input · P2 System Administration · P3 User · P4 Interface — which port carries the highest blast radius if a pairing is forged, and how is it isolated? If the permissions store records pairings rather than roles, what is the rollback when a device is deauthorized — does its access disappear at the next request or the next session? For a regulated investor: how does a pairing-based store satisfy "show me who had access on this date" without a password database to subpoena?
