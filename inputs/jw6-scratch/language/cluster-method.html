Co-location — Cluster Math

Cluster Methodology

A co-location cluster is defined by degree and distance — the rules that turn a scatter of stores into a ranked site-selection signal.
Degree tiers
01Primary target — the anchor the cluster is built around (e.g. a Walmart Supercentre).
02Secondary / tertiary targets — large-format and civic anchors that reinforce the primary draw.
03First-degree cluster — anchors within 1.0 km of the primary target.
04Up to fifth-degree cluster — the catchment expanded out to a 5.0 km threshold.
What the rules produce
✲The two thresholds — 1.0 km first-degree, 5.0 km fifth-degree — are the whole sorting rule. Every candidate site is scored by how many target tiers fall inside each ring.
✲Tiles for each target type — Walmart, Home Depot, Costco, hospital, higher-education — feed an 11-row ranking matrix with primary / secondary / tertiary weighting.
✲The output is a cluster record with a sales-per-square-foot proxy, ready to join against catchment and spend layers in the GIS engine.
✲Because the rule is purely geometric, it is reproducible and auditable — no opaque model weights stand between the input geography and the ranked output.
