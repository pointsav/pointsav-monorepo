Design System — Totebox OS

Totebox Services  ⇒  Containers

A Totebox is a micro or nano VM that runs each of its services in a separate container, so one box holds the full set of business records.
Totebox
Totebox Services
✲ a micro vm or nano vm
✲A Totebox runs different services, each isolated in its own container.
✲The services need not be transferable — only the Totebox itself ports between hosts, carrying its records with it.
Ex Firecracker (AWS) — a micro VM that still runs containers inside, the same isolation pattern applied here.
