# Workplace PDF

Sovereign desktop PDF viewer and print tool.
Opens PDF files using the pdfium-render crate (Apache 2.0 — Google PDFium).

**Platform:** macOS 10.13 High Sierra and later  
**Stack:** Tauri v1.7 + pdfium-render  
**Licence:** Apache-2.0

## Wave 2 — foundation scaffold

This application is in Wave 2 (foundation committed 2026-05-27).
Active development begins after Wave 1 trio ships.

## Planned capabilities

- Open and navigate PDF files page by page
- Print to PDF via OS print dialogue
- Zoom, pan, text selection
- Configurable endpoint for document retrieval from Foundry services
