# Workplace PDF

Visor de PDF de escritorio soberano y herramienta de impresión.
Abre archivos PDF usando el crate pdfium-render (Apache 2.0 — Google PDFium).

**Plataforma:** macOS 10.13 High Sierra y versiones posteriores  
**Tecnología:** Tauri v1.7 + pdfium-render  
**Licencia:** Apache-2.0
