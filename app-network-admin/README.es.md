# app-network-admin

Aplicación de plano de control para `os-network-admin` — la autoridad de enrutamiento de malla PPN.

Ejecuta en `os-network-admin`. Desplegado como `route-network-admin` en el nivel de
presentación y `route-network-admin-1` en Laptop A dentro de la Red Privada de PointSav.

Gestiona la configuración de malla WireGuard y publica el estado en vivo en
`public/mesh-state.json`.
