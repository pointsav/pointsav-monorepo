## Resumen

`system-core` es el cajón de primitivas de datos para **El Sustrato
del Libro Mayor de Capacidades** (afirmación doctrinal #33): el
estado de capacidades del sustrato ES el libro mayor WORM. Este
cajón define los tipos `Capability` y `WitnessRecord` contra los
que se vinculan todos los demás cajones `system-*` y `moonshot-*`.

**Estado:** Activo — Fase 1A (primitivas del Sustrato de Libro
Mayor de Capacidades).

**Ancla constitucional:** Afirmación #33 de DOCTRINE.md.
**Especificación operativa:**
`conventions/system-substrate-doctrine.md` §3.1 + §5.

La documentación técnica detallada — esquemas de tipos, lista de
cosas que el cajón NO contiene, instrucciones de compilación y
prueba — vive en el documento canónico en inglés
([README.md](./README.md)) por convención bilingüe del espacio de
trabajo (`~/Foundry/CLAUDE.md` §6: el español es panorámica
estratégica, no traducción literal).
