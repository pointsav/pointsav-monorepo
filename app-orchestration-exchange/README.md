## I. ARCHITECTURAL MANDATE

`app-orchestration-exchange` is the browser-based ad campaign management UI
for the Leapfrog 2030 reverse-flow substrate. It provides campaign creation,
bid-strategy configuration, and reporting against the `service-exchange`
backend. Deployed as `gateway-orchestration-exchange-N`. Runs on
`os-orchestration`.
