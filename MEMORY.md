# Memory — Mathew's Sandbox

Persistent memory for sandbox-rooted Claude sessions lives at:

```
~/.claude/projects/-home-mathew-sandbox/memory/
```

Claude Code auto-creates that directory on first session. The
session itself maintains the entries — write `user`, `feedback`,
`project`, and `reference` memories there as the conversation
warrants, and keep `MEMORY.md` inside that directory as the index.

This file (`/home/mathew/sandbox/MEMORY.md`) is just a pointer for
you-the-human to remember where memory lives. The actual memory
files belong inside `~/.claude/projects/-home-mathew-sandbox/memory/`,
not here.
