## Qué reemplaza

Este stub rastrea la intención a largo plazo de reemplazar **CodeMirror 6**
(JS vendorizado, ~100KB minificado, licencia MIT) con un componente de
edición de código interno diseñado específicamente para el flujo de trabajo
de edición de esquemas DTCG/PBS-1.

CodeMirror 6 se utiliza en `app-privategit-bim` como el panel "Code" del
editor de modo dual de DTCG (alternancia Visual / Code).

## Horizonte temporal

**Horizonte largo (2029+).** No comenzar la implementación hasta que
`app-privategit-bim` v1 esté en producción y `moonshot-schema-validator`
esté disponible (la integración de lint WASM es el prerequisito).
